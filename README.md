<h2><a class="anchor" id="overview"></a>Overview</h2>
This project forecasts stock prices using time series models (ARIMA, SARIMA, STL) and deep learning neural networks (LSTM, GRU).
It integrates technical indicators like EMA, MACD, RSI, and Stochastic Oscillator to enrich predictions.
Models are benchmarked with RMSE, MAE, RÂ², and MAPE to ensure accuracy and reliability.
The solution is deployed via FastAPI, Docker, and Streamlit dashboards, enabling scalable, interactive financial insights.
